      Hi,

      If you want call function of javascript object and return value using mshtml,I suggest you to add HTML field  in your HTML file and modified the  JavaScript source code to save the timeslots,So that we can capture it .

      Modify your HTML document's body so that it contains a hidden input field with an id attribute of 'result'

      <input type='hidden' id='result' value='' />

      Add the code in your JS function:

      document.getElementById('result').value = timeslots;

      And In you C# code:

      Result = document.getElementById(”result”).getAttribute(”value”);

      By the way ,Like Molke said, if you can also use the HtmlDocument.InvokeScript Method to achive your goad:

      HtmlDocument doc = webBrowser1.Document;
String str = doc.InvokeScript("timeslots ").ToString() ;
